1.创建仓库
  $ git init
2.添加文件a.txt到仓库并提交
  $ git add a.txt
  $ git commit -m "wrote a a.txt file"
  $ git remote add origin git@github.com:tlikai/vimrc.git
  $ git push origin master 提交本地修改到远程仓库master分支
3.查看仓库当前的状态
  $ git status
4.查看文件a.txt做了什么修改
  $ git diff a.txt
5.显示从最近到最远的提交日志
  $ git log
6.版本退回上一个版本
  $ git reset --hard HEAD^
7.版本退回上上一个版本
  $ git reset --hard HEAD^^
8.版本退回指定版本
  $ git reset --hard 3628164
9.从远程库克隆
  $ git clone git@github.com:michaelliao/gitskills.git
  $ git clone https://github.com/GreedyGrand/cainiao.git
10.分支
  a.创建分支dev，并切换到dev分支
  $ git checkout -b dev
  Switched to a new branch 'dev'
  ##
   git checkout命令加上-b参数表示创建并切换，相当于以下两条命令
   $ git branch dev
   $ git checkout dev
   Switched to branch 'dev'
  ##
11.处理冲突
  手动解决冲突后再提交。git status也可以告诉我们冲突的文件
  再提交：
  $ git add a.txt 
  $ git commit -m "conflict fixed"
  查看合并情况
  $ git log --graph
  最后删除分支
  $ git branch -d feature1
12.注释
  $ git commit -m "wrote a a.txt file"
  Added ( 新加入的需求 )
  Fixed ( 修复 bug )
  Changed ( 完成的任务 )
  Updated ( 完成的任务，或者由于第三方模块变化而做的变化 )
